# Loan Approval Analysis Dashboard

## Objective

Analyze loan application data to identify factors influencing loan approval decisions.

## Tools

- Power BI
- Excel

## KPIs

- Approval Rate
- Rejection Rate
- Applicant Segmentation
- Loan Amount Analysis

## Insights

- Identified key factors affecting loan approvals.
- Analyzed applicant demographics and financial characteristics.
- Developed interactive dashboards for decision support.
